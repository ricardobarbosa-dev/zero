Conceito — Como o Tailwind Funciona

O Tailwind é um framework utility-first: em vez de escrever CSS separado, você aplica classes diretamente no HTML. Cada classe faz uma coisa só.

❌ CSS Tradicional

/* styles.css */ .botao { background: blue; color: white; padding: 8px 16px; border-radius: 6px; font-weight: bold; }

✅ Tailwind CSS

<!-- HTML com Tailwind --> <button class="bg-blue-500 text-white px-4 py-2 rounded-md font-bold"> Clique </button>

⚙️ Setup Rápido (CDN para estudos)

<head> <!-- Adicione no <head> do seu HTML --> <script src="https://cdn.tailwindcss.com"></script> </head> <!-- Para produção, use o CLI (gera CSS otimizado): --> # npm install -D tailwindcss # npx tailwindcss init

Responsividade — Breakpoints Mobile-First

No Tailwind, você começa pelo mobile e adiciona prefixos para telas maiores. Sem prefixo = mobile. Com prefixo = a partir daquele breakpoint.

PrefixoLargura mínimaDispositivo
(sem prefixo)0px📱 Mobile (base)
sm:640px📱 Mobile grande / tablet pequeno
md:768px📟 Tablet
lg:1024px💻 Laptop
xl:1280px🖥️ Desktop
2xl:1536px🖥️ Tela grande
<!-- LEITURA: mobile=col-1, tablet=col-2, desktop=col-3 --> <div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3"> <!-- Texto: pequeno no mobile, grande no desktop --> <h1 class="text-2xl md:text-4xl lg:text-6xl">Título</h1> <!-- Esconder/mostrar por breakpoint --> <nav class="hidden md:flex">Menu Desktop</nav> <button class="md:hidden">☰ Menu Mobile</button> <!-- Padding responsivo --> <section class="px-4 md:px-8 lg:px-16 py-12 md:py-20">

Dicas & Truques Avançados

🎯 Centralizar qualquer coisa

<!-- Método 1: Flex --> <div class="flex items-center justify-center"> <!-- Método 2: Absolute center --> <div class="absolute top-1/2 left-1/2 -translate-x-1/2 -translate-y-1/2"> <!-- Método 3: Grid --> <div class="grid place-items-center">

🌙 Dark Mode

<!-- dark: aplica no modo escuro --> <div class="bg-white dark:bg-gray-900 text-gray-900 dark:text-white"> <!-- No tailwind.config.js: --> module.exports = { darkMode: 'class', // ou 'media' }

📦 Container padrão

<!-- Padrão que você vai repetir MUITO: --> <div class="max-w-7xl mx-auto px-4 md:px-8"> <!-- Conteúdo --> </div> <!-- max-w-7xl = 1280px máx --> <!-- mx-auto = centraliza --> <!-- px-4 md:px-8 = padding responsivo -->

Sua Ordem de Estudo

Semana 1–2: Fundamentos

Espaçamento (p, m, gap) → Tipografia → Cores → Display (flex, grid)

Semana 3: Layouts

Flexbox completo → Grid → Responsividade com prefixos (sm:, md:, lg:)

Semana 4: Componentes

Navbar, cards, botões, formulários, badges → Hover, focus, transições

Semana 5–6: Projeto real

Landing page completa → Dark mode → Otimização com CLI → Integrar com Django/Flask/FastAPI

Avançado: Configuração e Plugins

tailwind.config.js → Temas customizados → Plugins (@tailwindcss/forms, typography) → Purge CSS
